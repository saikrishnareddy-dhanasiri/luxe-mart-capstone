# 🛍️ LuxeMart - Premium E-commerce Experience
**Frontend Development Capstone Project**

LuxeMart is a minimalist, high-end e-commerce platform designed for the modern shopper. This project demonstrates a complete frontend architecture using **React**, **Redux**, and **Tailwind CSS**, focusing on clean aesthetics, state management persistence, and a seamless user experience.

---

## 🚀 Live Demo
**[View Live Project](https://luxe-mart-capstone.vercel.app/)**

---

## ✨ Key Features

- **Dynamic Product Dashboard**: Fetches real-time data from the FakeStore API.
- **Advanced Filtering**: Instant search functionality and category-based filtering (Electronics, Jewelry, Clothing).
- **Persistent Shopping Cart**: Add multiple units of the same product, update quantities, and remove items with data persisted via LocalStorage.
- **Wishlist System**: Save favorite items for later with a dedicated favorites management page.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Form Validation**: Secure login portal with real-time email and password validation.
- **Luxury UI/UX**: Includes a custom auto-sliding hero carousel, product hover effects, and a detailed FAQ accordion.

---

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/) (Vite)
- **State Management**: [Redux](https://redux.js.org/) with [Redux-Thunk](https://github.com/reduxjs/redux-thunk)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Routing**: [React Router Dom v6](https://reactrouter.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Testing**: [Vitest](https://vitest.dev/) & [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)
